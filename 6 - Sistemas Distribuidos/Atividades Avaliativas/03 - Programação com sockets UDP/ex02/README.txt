- Sistema Operacional Utilizado:
    Windows 11


- Como compilar/executar:
    $ python server.py
    $ python client.py


- Como utilizar:
    1. Abra dois terminais T1 e T2

    2. No T1, execute o servidor: $ python server.py

    3. No T2, execute um cliente: $ python client.py
    4. No T2, digite o caminho de um arquivo
